# 🍽️ SustainAI – Food Waste AI Management System

SustainAI is an AI-powered web application designed to reduce food waste by connecting hotels with NGOs. It enables efficient food redistribution using intelligent features like expiry prediction, route optimization, and smart suggestions.

## 🚀 Features

- 🔐 User Authentication (Admin, Hotel, NGO)
- 🍛 Food Item Management (Add, View, Upload Images)
- ⏳ AI-Based Expiry Prediction
- 🤖 AI Suggestions for Food Utilization
- 📦 NGO Request System
- 🚚 Delivery Management
- 🗺️ A* Route Optimization for Delivery
- 💬 AI Chatbot Support
- 📊 History Tracking
